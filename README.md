# Trading Bot with NLP and Time Series

Trading bot that combines NLP (news sentiment analysis) with time series forecasting and model input security.

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Usage

```bash
python -m src.main
```
